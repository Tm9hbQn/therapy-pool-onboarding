# Workflow: יצירת פרויקט בסיס (Initial Setup)

## מטרה
הקמת סביבת הפיתוח הראשונית של הפרויקט מבוססת React ו-Vite.

## שלבי ביצוע
1. פתח מסוף (Terminal) בתיקיית העבודה הראשית (`c:\Users\lavno\OneDrive\Desktop\projects\poolclient`).
2. הרץ את הפקודה: `npx -y create-vite@latest . --template react-ts` או `react` רגיל. הערה: יש לאשר את אזהרת מחיקת הקבצים הקיימים (Vite מתריע על תיקייה שאינה ריקה) או לחילופין להריץ לתוך תיקייה זמנית ולהעתיק. *המלצה*: להתקין ידנית `npm i react react-dom` יחד עם Vite אם נוצר קונפליקט עם קבצים קיימים.
3. הרץ `npm install` להתקנת תלויות.
4. נקה את תיקיית `src` מקבצי הדיפולט המיותרים של Vite (כגון לוגו, css ישן).
5. ודא שהאפליקציה עולה בעזרת `npm run dev`.
6. עדכן את `SESSION_LOG.md` כי שלב זה הסתיים.

## וידוא ותוצר
- קיומו של קובץ `package.json` תקין עם הפעלות סטנדרטיות.
- ריצת בסיס ללא שגיאות ב-Console.
